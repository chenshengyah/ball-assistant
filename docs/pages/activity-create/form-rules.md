# 创建活动页字段规则

## 1. 发布身份

- 必选
- 默认选中上次使用身份或默认身份
- 若当前无可用俱乐部身份，展示“去注册俱乐部”

## 2. 活动标题

- 必填
- 长度 1~30

## 3. 活动日期

- 必填
- 不能早于今天

## 4. 活动时间

- 开始时间必填
- 结束时间必填
- 结束时间必须晚于开始时间

## 5. 场馆

- 必选
- 来自当前身份下可用场馆列表

## 6. 报名模式

- 必选
- 枚举：GENERAL / COURT_BASED

## 7. 场地选择

- 仅当 signupMode = COURT_BASED 时显示
- 至少选择一个场地
- 只能选择 status = ACTIVE 的场地

## 8. 收费方式

- 必选
- 枚举：FREE / FIXED

## 9. 收费金额

- 仅当 feeType = FIXED 时显示
- 必填
- 大于等于 0

## 10. 人数上限

- 必填
- 必须大于 0

## 11. 是否允许候补

- 默认 true

## 12. 活动说明

- 选填
- 最长 500 字
