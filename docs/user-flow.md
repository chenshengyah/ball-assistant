# 用户流程

## 1. 创建活动流程

创建活动页
-> 选择发布身份
-> 若无俱乐部身份，去注册俱乐部
-> 返回创建活动页
-> 填写活动信息
-> 选择场馆
-> 根据报名模式决定是否选择场地
-> 提交发布
-> 跳转活动详情页

## 2. 俱乐部注册流程

创建活动页点击“去注册”
-> 进入俱乐部注册页
-> 提交注册信息
-> 注册成功
-> 返回创建活动页
-> 自动选中新注册的俱乐部身份

## 3. 场馆/场地管理流程

创建活动页点击“管理场馆/场地”
-> 进入场馆列表
-> 新增/编辑场馆
-> 进入场馆详情
-> 新增/编辑/停用场地
-> 返回创建活动页
-> 可继续选择场馆/场地
