1. 先看全局文档，理解产品范围、流程和统一状态口径

4. 最后看归档文档，仅用于兼容旧链接或追溯历史说明

## 页面地图摘要

当前文档口径下，MVP 页面关系固定为：

- 首次触发关键动作但资料未完善：进入 `完善信息 / 资料补全`
- 已建档主办方：`创建活动` -> `活动详情` -> `我的活动（我发布的）`
- 已建档报名者：`活动列表` -> `活动详情` -> `我的活动（我报名的）`
- 创建活动前缺俱乐部主体：`创建活动` -> `俱乐部注册` -> 返回 `创建活动`
- 创建活动前缺场馆主数据：`创建活动` 内进入 `场馆与场地管理` -> 返回 `创建活动`
- `我的` tab：作为资料与入口容器，承接去 `完善信息 / 资料补全`、`俱乐部注册`、`我的活动`
