# 球局助手 MVP 总览

## 1. 产品目标

- 帮助俱乐部/组织者快速发布球局活动
- 帮助用户查看、报名、参与活动
- 补齐用户基础资料，统一昵称与性别资料口径
- 先跑通“游客浏览 -> 动作前登录补资料 -> 创建活动 / 报名活动”主链路

## 2. 目标用户

- 普通用户
- 俱乐部管理员 / 活动组织者

说明：

- 本文仅描述目标用户和 MVP 范围
- 发布身份与俱乐部成员权限边界以 [角色边界说明](./role-boundaries.md) 为准

## 3. MVP 范围

### 本期要做

- 完善信息 / 资料补全
- 发布身份确认
- 俱乐部注册
- 场馆管理
- 场地管理
- 创建活动
- 活动列表
- 活动详情
- 报名 / 取消报名
- 我的活动

### 本期不做

- 在线支付
- 聊天
- 评论
- 复杂统计
- 审批流
- 多层级权限系统

## 4. 核心页面

- 首页
- 完善信息 / 资料补全
- 活动列表
- 活动详情
- 创建活动
- 俱乐部注册
- 场馆/场地管理
- 我的

## 5. 核心业务原则

- 用户无需先建档即可浏览首页、活动列表、活动详情和“我的”游客态
- 进入创建活动、报名/取消报名、我的活动、俱乐部入口等角色相关动作前，必须完成基础资料补全
- 一个活动必须归属一个发布身份/俱乐部
- 场馆/场地是主数据
- 活动发布时保留必要快照
- 历史活动不受主数据后续修改影响
