# 用户流程

## 1. 用户建档流程

用户首次进入小程序
-> 进入用户建档页
-> 填写昵称
-> 选择性别（`MALE` / `FEMALE`）
-> 提交成功
-> 进入首页或来源页

## 2. 创建活动流程

创建活动页
-> 当前用户已完成基础建档
-> 选择发布身份
-> 若无俱乐部身份，去注册俱乐部
-> 返回创建活动页
-> 填写活动信息
-> 选择场馆
-> 根据报名模式决定是否选择场地
-> 提交发布
-> 跳转活动详情页

## 3. 俱乐部注册流程

创建活动页点击“去注册”
-> 进入俱乐部注册页
-> 提交注册信息
-> 注册成功
-> 返回创建活动页
-> 自动选中新注册的俱乐部身份

## 4. 场馆/场地管理流程

创建活动页点击“管理场馆/场地”
-> 进入场馆列表
-> 新增/编辑场馆
-> 进入场馆详情
-> 新增/编辑/停用场地
-> 返回创建活动页
-> 可继续选择场馆/场地
