# 用户流程

## 1. 用户建档流程

用户首次进入小程序
-> 可直接浏览首页 / 活动列表 / 活动详情 / 我的（游客态）
-> 用户点击创建活动 / 报名 / 取消报名 / 我的活动 / 俱乐部入口
-> 若资料未完善，则进入用户建档页
-> 填写昵称
-> 选择性别（`MALE` / `FEMALE`）
-> 提交成功
-> 返回来源页并继续原动作

## 2. 活动主办方流程

创建活动页
-> 当前用户已完成基础建档
-> 选择发布身份
-> 若无俱乐部身份，去注册俱乐部
-> 返回创建活动页
-> 填写活动信息
-> 选择场馆
-> 根据报名模式决定是否选择场地
-> 提交发布
-> 跳转活动详情页
-> 在详情页查看报名情况
-> 活动开始后自动进入进行中
-> 活动结束时间到后自动进入已结束
-> 回收到“我的活动-我发布的”

## 3. 活动报名者流程

活动列表或其他入口
-> 进入活动详情页
-> 查看活动信息与报名状态
-> 若未登录或资料未完善，则在点击报名时先进入用户建档页
-> 根据报名模式执行报名
-> 报名成功后留在活动详情页
-> 在可取消窗口内可取消报名
-> 活动开始后自动进入进行中
-> 活动结束时间到后自动进入已结束
-> 回收到“我的活动-我报名的”

## 4. 俱乐部注册流程

创建活动页点击“去注册”
-> 进入俱乐部注册页
-> 提交注册信息
-> 注册成功
-> 返回创建活动页
-> 自动选中新注册的俱乐部身份

## 5. 场馆/场地管理流程

创建活动页点击“管理场馆/场地”
-> 进入场馆列表
-> 新增/编辑场馆
-> 进入场馆详情
-> 新增/编辑/停用场地
-> 返回创建活动页
-> 可继续选择场馆/场地

## 6. 主办方最小管理动作流程

活动详情页
-> 主办方查看活动
-> 根据需要执行调场 / 调容量 / 取消活动 / 再次发布
-> 动作结果继续留在活动详情页或进入再次发布页

说明：

- 活动详情页是活动发布后的主承接页面
- 首页、活动列表、活动详情、“我的”tab 都支持游客态浏览
- 活动结束按时间自动判定，不提供手动结束
- “我的活动”负责活动回收，不替代详情页
