# ball-assistant

微信小程序项目，使用 TypeScript 开发，并保留云开发初始化能力。

## 开发说明

- 小程序源码目录：`miniprogram/`
- 产品与页面文档目录：`docs/`
- 微信开发者工具已启用 `useCompilerPlugins: ["typescript"]`
- 当前包含自定义 tabbar 和三个基础页面：`首页`、`活动`、`我的`
